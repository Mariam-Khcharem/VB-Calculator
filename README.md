A simple calculator built with Visual Basic .NET and Windows Forms.
Features:
- Basic math operations (+, -, ×, ÷)
- Keyboard and mouse support
- Error handling
- Multi-line display
How to Run:
1. Open in Visual Studio
2. Press F5 to run
